JavaScript relationship with HTML and CSS

Building web application is like making an interactive children book. The author is the HTML, who provide the content and structure of the story, The illustrator is the CSS, who provide the styling including color, sizing of the pictures, the layout of the pages. JavaScript is the modern technologies that make the book comes alive, be it opening of a page or touch of a button on a page of the book.

Document Object Model, the DOM

DOM provides an interface on a browser for web developers to use JavaScript to interact with other documents such as HTML. In the DOM, every HTML element is represented as objects. Web developers could then use programming language such as JavaScript to access any elements in order to modify or create elements on a browser. These codes could either be written in the HTML file or a link to a seperate JavaScript file within the 'script' tags.
