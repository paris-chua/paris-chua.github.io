JavaScript relationship with HTML and CSS

Building web application is like making an interactive children book. The author is the HTML, who provide the content and structure of the story, The illustrator is the CSS, who provide the styling including color, sizing of the pictures, the layout of the pages. JavaScript is the modern technologies that make the book comes alive, be it opening of a page or touch of a button on a page of the book.

Document Object Model, the DOM

DOM provides an interface for web developers to use JavaScript on a browser to interact with other documents such as HTML. In the DOM, every HTML element is represented as objects. JavaScript could then be used to access any elements in order to modify or create elements on a browser.

Arrays and Objects

An array is a variable that could hold more than one value. An object could also be a variable that could hold more than one value (aka member). This member is a data type with name (key) and value/ values pair called property. Accessing data from them requires different structure of statements or syntax. For an array, we uses number (index) to access its data. First item in an array is indexed as 0. For example, let animals = ['dogs', 'cats', 'mice', 'rabbits'} animals[0] = 'dogs' animals[3] = 'rabbits' For an object, we uses name or key to access its data. For example, let person = { lastName: 'Chua', firstName: 'Paris', hairColor: 'Black', eyeColor: 'Dark Brown' } person.hairColor = 'Black'
